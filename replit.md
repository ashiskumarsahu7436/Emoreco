# EMORECO - Voice-Based AI Emotion Recognition

## Overview
EMORECO is a single-page web application that demonstrates voice-based emotion recognition using AI. The application provides an interactive interface for users to either speak live through their microphone or upload audio files to analyze emotions.

## Project Structure
- `index.html` - Main application file containing HTML, CSS, and JavaScript
- Static web application with no backend dependencies

## Features
- **Live Microphone Analysis**: Real-time emotion detection from voice input
- **Audio File Upload**: Analyze emotions from uploaded audio files
- **Interactive UI**: Modern, responsive design with visual audio feedback
- **Web Audio API**: Uses browser's Web Audio API for microphone access and visualization

## Technology Stack
- Pure HTML/CSS/JavaScript
- Font Awesome icons
- Web Audio API for audio processing
- Responsive design for mobile and desktop
